Módulo de Gestión de Torneos ()
## Entidades y Características * Torneo (Tournament) * idTorneo: UUID (Identificador único) * nombre: String * fechaInicio: Date * fechaFin: Date * lugar: String * tipoTorneo: Enum (Ej: INDIVIDUAL, DOBLES, EQUIPOS) * formato: FormatoTorneo (Referencia a una entidad/estrategia para el formato, ej: Eliminatoria Directa, Round Robin) * estado: Enum (Ej: PLANIFICADO, INSCRIPCION_ABIERTA, EN_CURSO, FINALIZADO, CANCELADO) * reglasEspecificas: String (Opcional) * organizador: Usuario (Referencia al usuario organizador) * Relaciones: * Tiene muchos ParticipanteInscrito * Tiene muchas Ronda (o Grupo si es Round Robin) * Gestionado por un Organizador (Usuario) * FormatoTorneo (TournamentFormat) (Podría ser una interfaz/clase abstracta con implementaciones concretas) * idFormato: UUID * nombreFormato: String (Ej: "Eliminatoria Directa", "Doble Eliminatoria", "Round Robin") * descripcion: String * Métodos (conceptual): generarCuadro(), determinarSiguienteRonda()
