FormCommentThis form illustrates the Form plugin for Archi. It creates controls showing up some Archi components' properties. It replaces the description field by the value of property "description_xxx" if it exists, where "xxx" is the value of the model's property "language". For instance, if the model has got a "language" property with value "fr", then the "documentation" field will be replaced by the value of the property "documentation_fr" if any.
